## Запуск
1. Сборка проекта:
   ```bash
   ./gradlew build
   ```
2. Исполнение:
   ```bash
   java -jar build/libs/lab5-RELEASE.jar
   ```

## Готовый JAR
Скачайте [lab5-RELEASE.jar](./build/libs/lab5-RELEASE.jar) и запустите:
```bash

```

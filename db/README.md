## Исполнение внутри интерактивной консоли psql
1. Создание таблицы:
   ```psql
   \i database.sql
   ```
2. Заполнение данными:
   ```psql
   \i data.sql
   ```
3. Удаление таблицы:
   ```psql
   \i drop.sql
   ```
